最终项目结构

src/main/java/org/example/                                                                                                                                                                                                                                                                                     
├── Main.java                          # 入口，解析参数，驱动输出                                                                                                                                                                                                                                                
├── model/                                                                                                                                                                                                                                                                                                       
│   ├── MappingType.java               # 枚举：ATOMIC / COMPOSITE / PARAMETERIZED
│   ├── FieldRef.java                  # 单个字段引用：ClassName.fieldName                                                                                                                                                                                                                                       
│   ├── ExpressionSide.java            # equals 一侧的字段集合 + 算子描述                                                                                                                                                                                                                                        
│   ├── FieldMapping.java              # 一条完整映射关系                                                                                                                                                                                                                                                        
│   └── ClassRelation.java             # 两个类之间的所有映射聚合                                                                                                                                                                                                                                                
├── analyzer/                                                                                                                                                                                                                                                                                                    
│   ├── JavaFileScanner.java           # 递归扫描 .java 文件                                                                                                                                                                                                                                                     
│   ├── FieldRefExtractor.java         # 从表达式提取字段引用 + 算子识别                                                                                                                                                                                                                                         
│   └── LineageAnalyzer.java           # Pipeline 协调器                                                                                                                                                                                                                                                         
├── visitor/                                                                                                                                                                                                                                                                                                     
│   ├── EqualCallSite.java             # 单个 equals() 调用点数据                                                                                                                                                                                                                                                
│   └── EqualsCallVisitor.java         # AST Visitor，收集所有 equals 调用                                                                                                                                                                                                                                       
├── classifier/                                                                                                                                                                                                                                                                                                  
│   └── RelationshipClassifier.java    # 三类关系判别逻辑                                                                                                                                                                                                                                                        
├── graph/                                                                                                                                                                                                                                                                                                       
│   └── LineageGraph.java              # 聚合 FieldMapping → ClassRelation                                                                                                                                                                                                                                       
└── renderer/                                                                                                                                                                                                                                                                                                    
├── MermaidRenderer.java           # Mermaid flowchart 输出                                                                                                                                                                                                                                                
└── TableRenderer.java             # 文本表格输出（支持中文宽度对齐）

使用方式

java -jar classRelation-1.0-SNAPSHOT-jar-with-dependencies.jar /path/to/your/project
